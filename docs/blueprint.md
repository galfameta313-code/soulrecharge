# **App Name**: Soul Recharge

## Core Features:

- Retreat Information Display: Present comprehensive details about the 'Soul Recharge' retreat, including program benefits, daily flow, serene location (Bogor villa), and participant testimonials.
- Personalized Reflection Prompt Tool: Generate customized reflection questions and journaling prompts based on user input, acting as a preparatory tool to help individuals focus on their intentions before the retreat.
- Direct Contact & Reservation System: Provide clear calls-to-action for users to contact retreat organizers via WhatsApp or email for inquiries and reservation booking, leading to a personal consultation.
- Responsive Navigation: Implement an intuitive and responsive navigation system for seamless access to different sections of the retreat information, adaptable for both desktop and mobile devices.
- Dynamic Content Loading: Apply subtle, scroll-triggered fade-in animations to content sections as they become visible in the viewport, enhancing user engagement and guiding attention.

## Style Guidelines:

- Color Palette: Drawing inspiration from concepts of nature, serenity, and clarity, the app utilizes a tranquil light theme.
- Primary color: A deep and grounding Forest Green (#1F4726), selected to evoke nature and provide strong contrast for headings and important text against a light background.
- Background color: A soft and ethereal Mint Beige (#F0F3F0), which is a highly desaturated, bright hue from the green family, providing a clean and calming canvas.
- Accent color: A verdant and fresh Green (#6CB34D), serving as a complementary hue to the primary. This brighter green will be used for interactive elements, highlights, and icons to draw attention.
- Headline font: 'Playfair Display' (serif), chosen by the user for an elegant, high-end feel in titles and featured quotes.
- Body text font: 'Inter' (sans-serif), chosen by the user for its modern, neutral, and highly readable qualities in paragraphs and general content. Note: currently only Google Fonts are supported.
- Minimalist line-art SVG icons will be employed to represent concepts and services, enhancing visual clarity without clutter, often tinted with the accent color.
- A clean, section-based layout design that emphasizes visual breathing room. Content is presented within wide, centered containers, ensuring a clear information hierarchy and responsiveness across various screen sizes.
- Subtle scroll-triggered fade-in and translateY animations for content blocks create a smooth and engaging visual flow. Interactive elements like buttons and navigation links incorporate hover effects for clear user feedback.
